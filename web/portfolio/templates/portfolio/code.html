As well as many filters using regular and convolution kernels. MedBot

MedBot is an AI chat bot which tells the user if any of the drugs they're taking have adverse reactions. Furthermore, it can converse about general topics, due to a custom 4 million word corpus which I helped build. The drug interactions are analyzed using the Drug Interaction API provided by data.gov. Natural language processing and response construction is achieved through the NLTK library. The bot learns over the corpus in order to respond naturally and be able to converse about common subjects, and it also learns from past interactions with users. We created a website for the bot using Django and hosted it on AWS Elastic Beanstalk.

Continuous Authentication Android App

Seniors pursuing a B.S. in Computer Science can opt to take a class where a small group of students meets with a company in the industry to work on a project in a professional setting. Our company tasked us with creating a continuous authentication application to run on an android device. The application would run on devices carried by many workers at the company in order to save them time when logging in, as well as enhance security by detecting if the phone was stolen and ending the session. The app is written in Java and the authentication server uses spring. Authentication is achieved through OAuth using JWT tokens. The data that's used for authentication includes location, gait, scroll speed and more. A model is then created and learned over using TensorFlow and Keras.
